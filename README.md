# HM Support Client

Client-side WordPress plugin for Shop Manager to submit support tickets to **HM Support Hub**.

## Features
- Admin menu: HM Support
- Support form for Shop Manager
- Sends ticket to Hub via REST API
- Includes working hours notice + WhatsApp urgent link
- Settings: Hub URL, Site ID, API Key

## Setup
1. Install and activate the plugin on the client site.
2. Go to **HM Support → Settings**
3. Set:
   - Hub URL (default: https://hizlimagazapro.com)
   - Site ID
   - API Key
4. Use **HM Support → Support Form** to submit tickets.
